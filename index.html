🎬 動画10分分割ツール

📌 デモページについて

これは静的なデモページです。実際の動画処理機能を使用するには、ローカル環境でFlaskアプリケーションを起動してください。

ローカル環境でのみ動作します

✨ 主な機能

  • 📤 動画ファイル（MP4, MOV, AVI）のアップロード（最大10GB）
  • ✂️ 自動で10分ごとに動画を分割（最大3時間対応）
  • 📊 リアルタイムの処理進捗表示
  • 💾 個別ファイルまたはZIP一括ダウンロード
  • 🗑️ 24時間後の自動ファイル削除
  • 🌐 主要ブラウザ対応（Chrome, Firefox, Safari, Edge）

🛠️ 技術スタック

Python
Flask
MoviePy
HTML5
CSS3
JavaScript
FFmpeg

🏗️ システム構成

ユーザー → Webブラウザ → Flask Webサーバー → MoviePy（動画分割） → ファイルストレージ

🚀 ローカル環境でのセットアップ

以下のコマンドでローカル環境で実行できます：

# リポジトリをクローン git clone https://github.com/yaoo77/video-splitter-app.git cd video-splitter-app # 仮想環境を作成（推奨） python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate # 依存関係をインストール pip install -r requirements.txt # アプリケーションを起動 python app.py

起動後、ブラウザで http://localhost:5000 にアクセスしてください。
